# Heart Disease Classification using Machine Learning

## Project Overview

This project predicts the presence of heart disease using machine learning models trained on the UCI Heart Disease dataset.

## Models Used

* Logistic Regression
* Naive Bayes
* Decision Tree
* Random Forest

## Evaluation Metrics

* Accuracy
* Confusion Matrix
* Classification Report
* ROC-AUC Curve

## Results

* Logistic Regression AUC: 0.93
* Random Forest AUC: 0.93
* Naive Bayes AUC: 0.89
* Decision Tree AUC: 0.82

## Dataset

UCI Heart Disease Dataset

## Conclusion

Random Forest and Logistic Regression performed best with highest ROC-AUC scores.
